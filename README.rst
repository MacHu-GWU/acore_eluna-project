Install
------------------------------------------------------------------------------

``acore_eluna`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install acore-eluna

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade acore-eluna
